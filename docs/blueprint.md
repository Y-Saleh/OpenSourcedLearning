# **App Name**: OpenSourcedPractice

## Core Features:

- Problem Sets: Curated sets of coding challenges across various difficulty levels and algorithm categories.
- Code Editor: Built-in code editor with syntax highlighting and customizable themes.
- Local Judge: Lightweight evaluation of code submissions against predefined test cases. Provides immediate feedback.
- Progress Tracking: Offline persistence using IndexedDB to track progress, submissions, and lessons locally on the user's device.
- Solution Analyzer: Suggest optimal solution after the third failed attempt.
- Personalized Learning: Dynamically adjust problem difficulty based on user performance. Recommends next best topic to learn, acting as a study tool to tailor the learning path.

## Style Guidelines:

- Primary color: Deep sky blue (#3399FF) to create a calm and focused environment.
- Background color: Light grayish-blue (#E6F0FF) to reduce eye strain during extended coding sessions.
- Accent color: Soft lime green (#CCFF99) to indicate successful code evaluation.
- Body and headline font: 'Inter' (sans-serif) for clear readability in the interface and code editor.
- Code font: 'Source Code Pro' (monospace) for a distinct and legible code display.
- Use minimalistic line icons to represent various problem categories and functionalities.
- Subtle animations during code evaluation and feedback display for enhanced user experience.
